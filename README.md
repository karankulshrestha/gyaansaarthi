# Gyaan Saarthi App 🤠

"Gyaan Saarthi" is a transformative initiative addressing the knowledge gap in rural India. Utilizing OpenAI GPT models, it empowers individuals with limited literacy skills by providing essential information in their native languages. The platform is designed to break down language barriers and enhance access to vital documents, enabling users to make informed decisions independently.

## Key Features: 🤖

- **Precision in Native Language:** Clear and precise information delivered in users' native languages.
- **Voice Assistant Bot:** An intuitive bot facilitates effortless interaction, allowing users to ask questions and receive instant responses in their native language.
- **Document Assistance:** Users can upload and scan documents, seeking details and understanding independently with the help of the assistant bot.
- **Empowering Communities:** Beyond information, "Gyaan Saarthi" fosters community empowerment by disseminating knowledge about government policies and opportunities.
- **Education and Awareness:** Prioritizing education, the platform offers resources to enhance users' knowledge and skills, focusing on rights, available resources, and effective access strategies.


## Tech Stack

**App:** Flutter, Firebase Auth, Firestore Database, Multiple Libraries, Text to speech, Speech to Text.

**Backend:** FastAPI, Aws, CromaDB, Langchain, OpenAI, MongoDB.


## Launch the App

Clone the project

```bash
  git clone https://github.com/rajkulx/gyaansaarthi
```

Go to the project directory

```bash
  cd gyaansaarthi
```

Install dependencies

```bash
  flutter pub add
```

Start the app in Real device

```bash
  flutter run
```
